# 🤖 ТГ-Бот: Хранитель Контекста (Context Keeper)

Удобный инструмент для автоматизации ведения логов разработки и быстрого сохранения контекста по разным пет-проектам. Бот помогает фиксировать, на каком этапе остановилась работа, чтобы затем в один клик передавать этот контекст нейросетям (Gemini, Cursor, Claude).

---

## ✨ Основные возможности

* 🗂️ **Разделение по проектам:** Интерактивное меню позволяет переключать контекст между проектами (*Калькулятор арматуры* и *TaskFlow Checker*).
* 📝 **Мгновенное логирование:** Любое отправленное боту сообщение автоматически записывается в базу данных текущего проекта.
* 💾 **Локальная база данных:** Все записи сохраняются с точной датой и временем во встроенную БД `SQLite`.
* 🚀 **Экспорт контекста для ИИ:** По нажатию одной кнопки формирует готовый текстовый срез последних изменений, который можно скопировать и скинуть ИИ, чтобы тот мгновенно вошел в курс дела.
* 🔗 **Быстрый доступ:** В главном меню всегда доступны прямые ссылки на рабочие репозитории.

---

## 🛠️ Технологический стек

* **Язык:** Python 3
* **Фреймворк:** Aiogram 3 (асинхронная библиотека для Telegram-ботов)
* **База данных:** SQLite3
* **Безопасность:** `python-dotenv` для изоляции токенов и секретных ключей через `.gitignore`.

---

1. Клонировать репозиторий.
